# ExemploConsumer

Este projeto demonstra um consumidor RabbitMQ utilizando .NET 8 e a biblioteca RabbitMQ.Client.

## Download

Você pode baixar o executável **ExemploConsumer.exe** diretamente na página de releases do projeto:

[https://github.com/inovamobil/invm-dotnet-exemplo-consumer/releases/tag/v1.0](https://github.com/inovamobil/invm-dotnet-exemplo-consumer/releases/tag/v1.0)

## Detalhes do Código

Para visualizar e entender a implementação completa, acesse o arquivo [`Program.cs`](./ExemploConsumer/Program.cs) neste repositório.
